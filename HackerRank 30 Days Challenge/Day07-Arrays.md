# Day 7: Arrays

Task

Given an array, ***A***, of ***N*** integers, print ***A***'s elements in reverse order as a single line of space-separated numbers.

Details:

[This Link](https://www.hackerrank.com/challenges/30-arrays/problem)

**Sample Input**

```
4
1 4 2 3
```
**Sample Output**

```
3 2 4 1
```
